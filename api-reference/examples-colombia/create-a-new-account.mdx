In order to create a new colombian account, you will need to send both the colombian RUT FORM, and the test_set_id as an additional field,
 that you can get from the DIAN website. Remember to get the token from the /auth endpoint first.

## Endpoint: /accounts
* Method: POST
* Required headers:
  * Content-Type: application/json
  * Authorization: Bearer \<token\>
  * Accept: \*/\*
