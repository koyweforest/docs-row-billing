## Create new products
Products can be anything you buy or sell, it could be merchandise or services. A product can have an
available quantity on multiple warehouses and show different prices based on price lists.

* Endpoint: https://api-billing.koywe.com/V1/products
* Method: POST
* Required headers:
* Content-Type: application/json
* Authorization: Bearer \<token\>

## Create new product categories
Products can optionally belong to a category. Categories can also have a hierarchy by having a parent category.

* Endpoint: https://api-billing.koywe.com/V1/product_categories
* Method: POST
* Required headers:
* Content-Type: application/json
* Authorization: Bearer \<token\>
* Accept: \*/\*

```json
{
	"account_id": 39,
	"name": "categoria ejemplo API 3",
	"parent_product_category_id": null
}
```
